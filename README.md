### Description
Social network written with: react/redux/ts/antd/axios<br>

App functional:
- **Loginization**
- **CRUD**
- **handling errors**
- **Add / change an avatar in your profile**
- **Change your profile status**
- **Search friends and dialogs**
- **infinite scroll**
- **chat with friends**
- **pagination**
